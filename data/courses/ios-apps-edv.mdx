- iOS Developer Center, iOS Life Cycle
- Xcode (IDE), creación de proyectos
- Swift

  * Collection Types
  * Control Flow
  * Functions
  * Enumerations
  * OOP, Clases, Structs
  * Protocols, Clousures
- Versionamiento, Git, GitHub
- Xcode y el Interface Builder
- iOS Simulator
- iOS App Life Cycle
- UIKit
- Table Views & Picker View, Collection View
- View Controllers, Storyboards, Segues
- Multithreading
- WebServices con REST (JSON)
- Core Data & UserDefaults 
- CocoaPods y dependencias de terceros
