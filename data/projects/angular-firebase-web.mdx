Un blog, gestor de contenido, CMS desarrollado con Angular y apoyado con varios servicios de _Firebase_.

Soportaba la creación de _labs_, guías y tutoriales paso a paso.

## Website

[Live Preview](https://maksha-41f4f.web.app/)

## Tecnologías

Algunas librerías Angular y tools usadas:

- Angular CLI

  - [cli.angular.io](https://cli.angular.io/)
  - [github.com/angular/angular-cli](https://github.com/angular/angular-cli)

- Angular Material

  - [material.angular.io](https://material.angular.io/)
  - [github.com/angular/components](https://github.com/angular/components)

- AngularFire2
  - [github.com/angular/angularfire](https://github.com/angular/angularfire)

Firebase services usados:

- Realtime Database
- Authentication
- Storage
- Hosting
- Cloud Functions
