Un sitio desarrollado con Angular y apoyado con varios servicios de Firebase.

* Realtime Database
* Authentication
* Storage
* Hosting
* Cloud Functions

[Website](https://maksha-41f4f.web.app/)

<GitHub repo="maksha-web"/>

## Features

### Admin

En el apartado de administración se puede crear diferentes tipos de contenido y para el contenido de _Lab_ se pueden adicionar pasos. El contenido soporta formato markdown.

![Admin Panel 1](/images/projects/admin1.png)

![Admin Panel 2](/images/projects/admin2.png)


### Stats

Los usuarios se pueden autenticar para acceder a contenido exclusivo. El apartado de estadísticas muestra a los usuarios con un contador.

![Stats](/images/projects/stats.png)
